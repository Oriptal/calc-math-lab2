# Установка

Готовые сборки — на странице релизов:
**<https://git.oriptal.dev/cadmin/calc-math-lab2/releases>**

Скачайте файл под свою платформу:

| Платформа | Файл |
|---|---|
| **Linux** x86_64 (Intel/AMD) | `calc-math-lab2-linux-x86_64.tar.gz` |
| **Linux** ARM64 (aarch64) | `calc-math-lab2-linux-arm64.tar.gz` |
| **macOS** Intel **и** Apple Silicon (universal) | `calc-math-lab2-macos-universal.dmg` |

Приложение самодостаточное — Qt и прочие библиотеки уже внутри, ставить ничего дополнительно не нужно.

## macOS

> Приложение **не подписано** сертификатом Apple Developer (это платно), поэтому при первом запуске macOS его заблокирует. Это нормально — ниже как открыть. Сборка собирается автоматически из исходников в CI, внутри обычный Qt-бинарник.

1. Откройте скачанный `calc-math-lab2-macos-universal.dmg`.
2. В открывшемся окне перетащите **`comp-math.app`** в папку **`Applications`** (ярлык рядом).
3. Размонтируйте образ (значок диска на рабочем столе → «Извлечь»).

### Первый запуск (обход Gatekeeper)

При обычном двойном клике появится «**comp-math … cannot be opened because Apple cannot check it for malicious software**» (или «приложение повреждено»). Выберите **любой** из способов:

**Способ 1 — Терминал (самый надёжный, работает на всех версиях macOS).**
Снимает «карантин» с приложения:
```bash
xattr -dr com.apple.quarantine /Applications/comp-math.app
```
После этого запускайте двойным кликом как обычно.

**Способ 2 — Системные настройки.**
1. Попробуйте открыть приложение (появится предупреждение) — нажмите «Готово».
2. **Системные настройки → Конфиденциальность и безопасность**.
3. Внизу будет строка «**comp-math … было заблокировано**» → нажмите «**Открыть всё равно**» (*Open Anyway*).

**Способ 3 — правый клик (старые macOS до 15).**
Правый клик (Ctrl+клик) по `comp-math.app` → **Открыть** → в диалоге ещё раз **Открыть**.

Делается это **один раз** — дальше приложение открывается обычным способом.

## Linux

Требуется графическая среда (X11 или Wayland). Подойдёт любой современный десктоп-дистрибутив.

> ⚠️ **ARM64:** сборка требует **glibc ≥ 2.39** (Ubuntu 24.04+, Debian 13+, Fedora 39+, свежий Arch). Сборка x86_64 совместима шире — **glibc ≥ 2.35** (Ubuntu 22.04+).

1. Распакуйте архив:
   ```bash
   tar -xzf calc-math-lab2-linux-x86_64.tar.gz   # или -linux-arm64
   ```
2. Запустите:
   ```bash
   ./calc-math-lab2/bin/comp-math
   ```

Папку `calc-math-lab2/` можно положить куда угодно (например, в `~/.local/opt/`) и запускать оттуда — пути к библиотекам внутри относительные (`bin/qt.conf`).

### Ярлык в меню приложений (по желанию)

Создайте `~/.local/share/applications/comp-math.desktop`:
```ini
[Desktop Entry]
Type=Application
Name=Вычислительная математика
Exec=/абсолютный/путь/к/calc-math-lab2/bin/comp-math
Icon=/абсолютный/путь/к/calc-math-lab2/bin/comp-math
Categories=Education;Science;
Terminal=false
```
