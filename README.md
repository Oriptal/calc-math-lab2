# calc-math-lab2

Qt Quick application for ITMO computational mathematics lab 2.

## What It Does

- Solves a nonlinear equation with multiple numerical methods
- Solves a system of linear equations
- Provides a desktop UI built with QML
- Includes report sources in `doc/`

## Stack

- C++
- Qt / QML
- CMake

## Run

```bash
cmake -S . -B build
cmake --build build
./build/appHelloWorldQuickProject
```
